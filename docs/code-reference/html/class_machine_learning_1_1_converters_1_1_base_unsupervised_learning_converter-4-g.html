Подробное описание

Абстрактный класс базового конвертора данных в данные для предсказания без учителя и наоборот.

Наследует BaseLearningConverter<R, D, DT, DR>. Реализует IUnsupervisedLearningConverter<IEnumerable<double>, R, D, DT, DR>.

Параметры шаблона
RТип выходных данных для предсказания.
DТип данных.
DTТип входных данных.
DRТип выходных данных.
