Подробное описание

Абстрактный класс базового конвертора данных в данные для предсказания и наоборот.

Реализует ILearningConverter<IEnumerable<double>, R, DT, DR>.

Параметры шаблона
RТип выходных данных для предсказания.
DТип данных.
DTТип входных данных.
DRТип выходных данных.
